2. Lista ze Zdjęciem zamiast Punktora

Punktory są zastąpione obrazkiem PNG/SVG za pomocą CSS (`::before`).

  • Element z własnym, graficznym punktorem (ikona zaznaczenia).
  • Dzięki CSS można kontrolować rozmiar i pozycję obrazka.
  • To jest element trzeci.

3. Pozycjonowanie Punktora (`list-style-position`)

Ta lista ma ustawione `outside` (domyślnie). Punktor jest na zewnątrz bloku tekstowego.

  • Dłuższy tekst w elemencie listy. Gdy tekst zawija się do następnej linii, jest on wyrównany do lewej, a punktor pozostaje w swojej pierwotnej pozycji.

Ta lista ma ustawione `inside`. Punktor jest wewnątrz bloku tekstowego.

  • Dłuższy tekst w elemencie listy. Gdy tekst zawija się do następnej linii, zaczyna się bezpośrednio po punktorze, co tworzy większe wcięcie.

4. Lista z Całkowicie Usuniętym Punktorem

Użyto właściwości `list-style-type: none;` oraz zresetowano wcięcie.

  • Element bez żadnego symbolu punktora.
  • Idealne do tworzenia poziomych menu nawigacyjnych.
  • Element trzeci.

5. Skrócony Zapis Właściwości Punktora (`list-style`)

Zapis skrócony: `list-style: square inside none;`

  • Ustawienie typu (`square`), pozycji (`inside`) i obrazu (`none`) w jednej linii.
  • Upraszcza kod CSS i jest łatwiejsze do czytania.
  • Element trzeci.

6. Nadanie Kolorów Listom i Elementom Listy

Właściwość `color` nadaje kolor tekstu i punktorów. `background-color` tło bloków.

  • Element nieparzysty (tło czerwone, tekst czerwony, punktor czerwony).
  • Element parzysty (tło niebieskie, tekst niebieski, punktor niebieski).
